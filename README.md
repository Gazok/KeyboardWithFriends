Keyboard With Friends
=====================

A game made in 48 hours for [Ludum Dare](http://ludumdare.com) 28.

Theme: You Only Get One

[Entry Page](http://www.ludumdare.com/compo/ludum-dare-28/?action=preview&uid=6609)

Requirements
------------

Flash player

Binaries
--------

[Online](https://jamiebayne.co.uk/games/kwf/)

Build
-----

Requires the Flex SDK.

    git clone https://github.com/qualiaa/KeyboardWithFriends
    cd KeyboardWithFriends
    mxmlc src/Main.as
    flashplayer src/Main.swf
